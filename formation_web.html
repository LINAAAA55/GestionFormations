Formation : Développement Web

Apprenez à créer des sites et des applications web de A à Z. Cette formation couvre le développement front-end et back-end, de HTML/CSS jusqu'à PHP et les bases de données.

Informations générales

  • Durée : 8 semaines
  • Niveau : Débutant
  • Format : Présentiel
  • Langue : Français

Programme de la formation

Module 1 - HTML (1 semaine)

  • Structure d'une page web
  • Balises de base : titres, paragraphes, listes, liens, images
  • Formulaires HTML
  • Balises sémantiques (header, nav, section, footer)

Module 2 - CSS (2 semaines)

  • Sélecteurs et propriétés CSS
  • Mise en page : flexbox et grid
  • Couleurs, typographie et espacement
  • Design responsive (media queries)

Module 3 - JavaScript (2 semaines)

  • Variables, fonctions et conditions
  • Manipulation du DOM
  • Événements et formulaires dynamiques
  • Introduction à Ajax

Module 4 - PHP et bases de données (2 semaines)

  • Syntaxe PHP de base
  • Traitement des formulaires
  • Connexion à MySQL avec PDO
  • Opérations CRUD (Create, Read, Update, Delete)

Module 5 - Projet final (1 semaine)

  • Création d'un site web complet
  • Déploiement et présentation

Compétences acquises

À la fin de cette formation, vous serez capable de :

  • Créer des pages web en HTML et CSS
  • Ajouter de l'interactivité avec JavaScript
  • Développer des fonctionnalités back-end en PHP
  • Gérer une base de données MySQL
