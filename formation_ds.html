Formation : Data Science

La Data Science est au cœur de la transformation numérique. Cette formation vous apprend à collecter, analyser et visualiser des données pour en extraire des informations utiles à la prise de décision.

Informations générales

  • Durée : 10 semaines
  • Niveau : Intermédiaire
  • Format : Hybride (en ligne + présentiel)
  • Langue : Français

Programme de la formation

Module 1 - Python pour la Data (2 semaines)

  • Révision Python et environnement Jupyter
  • Manipulation de données avec Pandas
  • Calculs numériques avec NumPy

Module 2 - Bases de données et SQL (2 semaines)

  • Introduction aux bases de données relationnelles
  • Requêtes SQL : SELECT, JOIN, GROUP BY
  • Connexion Python - base de données

Module 3 - Statistiques appliquées (2 semaines)

  • Statistiques descriptives
  • Tests statistiques
  • Corrélation et régression

Module 4 - Visualisation des données (2 semaines)

  • Graphiques avec Matplotlib et Seaborn
  • Tableaux de bord interactifs avec Plotly
  • Bonnes pratiques de visualisation

Module 5 - Projet final (2 semaines)

  • Analyse complète d'un jeu de données réel
  • Rapport et présentation des résultats

Compétences acquises

À la fin de cette formation, vous serez capable de :

  • Nettoyer et préparer des données avec Pandas
  • Réaliser des analyses statistiques
  • Créer des visualisations claires et pertinentes
  • Présenter des résultats à des non-spécialistes
